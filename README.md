# HH Grade Analyzer

Firefox-расширение для автоматического определения уровня вакансий на hh.ru.

![Демонстрация](https://github.com/user-attachments/assets/9493c250-3003-401c-8c81-0dc81f3714f6)

## Возможности

- 🎯 **Автоопределение грейда**: Intern, Junior, Middle, Senior, Lead
- 📊 **Уверенность**: Процент точности определения (на основе 5000+ примеров)
- 📈 **Анализ зарплаты**: Сравнение с рыночными вилками
- 📝 **История**: Сохранение последних 100 просмотренных вакансий
- 🎨 **Визуальный индикатор**: Цветной бейдж на странице вакансии

## Установка

1. Скачай [hh_grade_analyzer-1.0.0.zip](https://github.com/Alex767-star/hh-grade-analyzer/raw/main/web-ext-artifacts/hh_grade_analyzer-1.0.0.zip)
2. Firefox → `about:addons` → Шестерёнка → "Установить дополнение из файла"
3. Выбери скачанный zip

## Использование

Открой любую вакансию на hh.ru — бейдж с уровнем появится автоматически.

## Как работает

Анализирует заголовок, описание, зарплату, опыт. База **5000+ паттернов** вакансий.

## Приватность

- Всё локально, никакие данные не уходят на сервер
- История только в твоём браузере
