# HH Grade Analyzer

Firefox-расширение для автоматического определения уровня вакансий на hh.ru.

## Возможности

- 🎯 **Автоопределение грейда**: Intern, Junior, Middle, Senior, Lead
- 📊 **Уверенность**: Процент точности определения (на основе 5000+ примеров)
- 📈 **Анализ зарплаты**: Сравнение с рыночными вилками
- 📝 **История**: Сохранение последних 100 просмотренных вакансий
- 🎨 **Визуальный индикатор**: Цветной бейдж на странице вакансии

## Установка

1. Скачайте архив репозитория
2. Откройте Firefox → `about:debugging`
3. Нажмите "Загрузить временное дополнение"
4. Выберите `manifest.json`

## Использование

Просто откройте любую вакансию на hh.ru — бейдж с уровнем появится автоматически.

## Приватность

- Все данные обрабатываются локально
- Никакая информация не отправляется на серверы
- История хранится только в вашем браузере
