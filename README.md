# HH Grade Analyzer

Firefox-расширение для автоматического определения уровня вакансий на hh.ru.

![Демонстрация](https://github.com/user-attachments/assets/9493c250-3003-401c-8c81-0dc81f3714f6)

## Возможности

- 🎯 **Автоопределение грейда**: Intern, Junior, Middle, Senior, Lead
- 📊 **Уверенность**: Процент точности с детальным объяснением
- 💬 **Объяснение**: Почему принято такое решение (заголовок, описание, ЗП, опыт)
- 📈 **Анализ зарплаты**: Сравнение с рыночными вилками
- 📝 **История**: Сохранение последних 100 просмотренных вакансий
- 🎨 **Визуальный индикатор**: Цветной бейдж на странице вакансии

## Установка

### Из XPI (рекомендуется)
1. Скачай [`hh_grade_analyzer-1.0.0.zip`](https://github.com/Alex767-star/hh-grade-analyzer/releases/latest/download/hh_grade_analyzer.xpi)
2. Firefox → `about:addons` → Шестерёнка → "Установить дополнение из файла"
3. Выбери скачанный `.xpi`

### Из исходников (разработка)
1. Клонируй репозиторий
2. Firefox → `about:debugging` → "Загрузить временное дополнение"
3. Выбери `manifest.json`

## Использование

Открой любую вакансию на hh.ru — бейдж с уровнем и объяснением появится автоматически.

## База паттернов (5000+ вакансий)

Классификатор обучен на агрегированных данных из открытых источников:

| Источник | Тип данных | Примеров |
|---|---|---|
| Парсинг hh.ru API | Заголовки вакансий, зарплаты, опыт | 3000+ |
| Хабр Карьера / LinkedIn | Уровни зарплат по грейдам | 500+ |
| Опросы зарплат (Habr, DOU, Glassdoor) | Рыночные вилки 2024-2025 | 1000+ |
| Ручная разметка | Ключевые слова и контекстные фразы | 500+ |
| ГОСТ / Профстандарты | Формальные требования к грейдам | 100+ |

**Методология**: данные агрегированы в паттерны (keywords, title_patterns, salary ranges, experience ranges) без хранения персональных данных. Никакие пользовательские данные не собираются.

## Публикация в Mozilla Add-ons (AMO)

Для публикации в официальном магазине Firefox:

1. Зарегистрируйся на [addons.mozilla.org](https://addons.mozilla.org/developers/)
2. Собери подписанный XPI:
   ```bash
   npm install -g web-ext
   web-ext sign --api-key=YOUR_KEY --api-secret=YOUR_SECRET --channel=listed
    Загрузи подписанный XPI в AMO

    Пройди ревью (обычно 1-3 дня)

Приватность

    ✅ Все данные обрабатываются локально

    ✅ Никакая информация не отправляется на серверы

    ✅ История хранится только в браузере

    ✅ Нет сбора телеметрии

    ✅ Нет доступа к cookies или сессии hh.ru
